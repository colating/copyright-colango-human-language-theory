# Markdown 文件的搜索方法

- 在 readthedocs.org网站上进行搜索，结果最精确。据说这是 rtd自己嵌入了的更强功能的搜索工具。
- 本地使用 RTD theme进行搜索，实测结果比其他 theme插件的搜索结果更精确。
- 使用其他的搜素语法，可提高搜索的精确度。
- 有些主题，使用 `Ctrl+K`快件将调出搜索菜单。

- 在 VSCode、或其他 Md编辑工具里进行搜索，也可以得到类似的最好结果？
