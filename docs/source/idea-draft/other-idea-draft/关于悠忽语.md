# 关于 colango

## colango体系    

### colating
colating是一种正在创造中的、原子概念标记号的总量尽可能多，以匹配到人类智慧无限创造能力极限、并具有强制性自我陈城代谢和自我否定之否定能力的通用型模糊语言文字，属于下一代拉丁语的范畴。

### cofang
cofang属于 100进制数学计数系统，是在创造 colating过程中所产生的衍生产品。为 IPv6域名，天文计量等提供支持。

### colango
colango是为创造 colating和 cofang提供理论支持的语言文字智慧理论思想体系。

## 为什么要对“道”和“白马非马”概念进行深入思考？

### “道”回答的是宇宙的极限，以及人类智慧能力的极限
宇宙的极限：绝对无穷，意味着概念对象的总量，也是绝对无穷的。   
人类智慧能力：相对无穷，意味着人类智慧的创造能力，也是相对无穷的。   
因此，Colango将 colating的原子概念总量，设计为尽可能的多，以匹配人类相对无限的创造能力。

### “白马非马”回答的是人与概念标记的相互影响问题。
概念标记是人类创造的，并且也是给人类自己使用的。因此，概念在使用过程中的实际释义，就必然会受到“人性”的影响。比如人类的思维方式等等。

## 为什么需要关注“人性”问题
概念标记号是人类创造的，并且也是给人类自己使用的。因此，概念在使用过程中的实际释义，就会有与人性有关的因素，通过各种方式进行介入。   
Colating关注和探讨人性，是为了：  
- 在构词法上，增加一些标记，或是其他方法来标识概念的合理精确度，以抑制概念级的悠忽度。   
- 在语法上，构建健硕的语法逻辑表达，以抑制句子或句子级别以上的悠忽度。  
- 从 Colango语言创造理论上，对保证 Colating语文表达的精确度作最后的努力。  
- 设定 Colating语文的强制轮转机制，就是从人性的角度来考量的。Colating既要保证在 4K年内提供提供足够的原子概念标记号；也要强制在 4K年左右的时候，就必须进行轮转，进行自我否定式的新陈代谢。
- 从理论上，不希望任何一种语言一家独大，因为每一种长期被使用的语言文字，都会产生一种创世纪的传说。任何一种一家独大的传说，都有可能让社区的思想陷入僵化。水至清而无鱼。
