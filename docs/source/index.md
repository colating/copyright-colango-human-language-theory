# Colango 语言文字理论 & 语文智慧学

## 关于 Colango                    

创极语是一种与眼、口、耳有关的、属于优选数值型、优选声节型概念标记号的人类通用语言文字。

### 创极语目前由 4个部分构成

1. Colating 人类通用语言文字：  
与人类现行各个语种的语言文字类似，copyleft colating 许可，任何人都可以无任何附加条件地使用。  
Colating 人类通用语言，许可要求不会高于、或多于任何一种联合国官方语言： [copyleft 许可](LICENSE) [link](https://github.com/colating/copyleft-colating-general-human-language/tree/tscross)  
2. colango 通用语言原理：  
致力于从最本质的角度来研究人类通用语言文字的原理，以及解释 Colating 人类通用语言文字在创造过程中的智慧学思想。[copyright 许可](LICENSE) [link](https://github.com/colating/copyright-colango-human-language-theory/tree/gscross)  
3. Colating IME 输入法  
目前的许可 取决于源头 [RIME输入法的许可  GPL-3.0](license https://github.com/rime/ibus-rime)，未来的目标是采用 MIT许可。

## colating language 构词法、语法

- 特点    
  - 有效的原子概念标记号的创造，永远掌握在概念创造者的手中。
  - 创极语概念标记号（相当于：英语单词）的理解与传播，可以不需要第三者的介入。
  - 易于理解，易于创造原子概念标记号。
  - 兼容在未来 AI 可能可以自行创造的概念标记号。
  - 创极语的创造，以无限语言哲学理论作为基础支撑。

### 创极语的权利要求

最小约束条件，为了保证 Colating Language 创造的顺利进行， Colating 字母采用 类似软件行业的 MIT许可，默认有 4种常用拉丁字重（Light‌，‌Regular、‌Bold 和 ‌Italic）的权利要求。

- Colating 人类通用语言的许可，不会高于、或多于任何一种联合国官方语言。但也不会超出不同于联合国官方语言中的不合理部分。  
- Colating 为了确保本语言文字能够被正确使用，会发布无版权要求的语法（含构词法）等的资料。  
- Colango (clgo-) 的理论创作作品，有版权要求,此收入作为 Colating开发的费用之一，未来期望有基金会进行管理。目前其他主流语言的思想作品，也是有版权要求的。
- 通用语言本身是没有版权要求的，而思想作品是有版权要求的，注意它们之间的区别：拉丁语语言本身是没有版权，任何人的可以使用它；而采用拉丁语写了一部流行小说，这部小说是有版权要求的。
- 其他，待完善。

注：英语印刷字体中，‌Regular（常规体）‌、‌Bold（粗体）‌ 和 ‌Italic（斜体）‌ 是基础字形变体，广泛应用于文档排版、设计规范和数字界面。Regular 为默认标准笔画，Bold 通过加粗突出视觉重心，Italic 则倾斜变形以增强动态感或强调特定内容 ‌。
