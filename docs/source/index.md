## 关于 Colango                    

创极语是一种与眼、口、耳有关的、属于优选数值型、优选声节型概念标记号的人类通用语言文字。

### 创极语目前由 4个部分构成

1. Colating 人类通用语言文字：  
与人类现行各个语种的语言文字类似，copyleft colating 许可，任何人都可以无任何附加条件地使用。  
Colating 人类通用语言，许可要求不会高于、或多于任何一种联合国官方语言： [copyleft 许可](LICENSE) [link](https://github.com/colating/copyleft-colating-general-human-language/tree/tscross)  
2. colango 通用语言原理：  
致力于从最本质的角度来研究人类通用语言文字的原理，以及解释 Colating 人类通用语言文字在创造过程中的智慧学思想。[copyright 许可](LICENSE) [link](https://github.com/colating/copyright-colango-human-language-theory/tree/gscross)  
3. Colating IME 输入法  
目前的许可 取决于源头 [RIME输入法的许可  GPL-3.0](license https://github.com/rime/ibus-rime)，未来的目标是采用 MIT许可。
