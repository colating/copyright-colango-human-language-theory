# Colango 概念理论

1. 概念标记号
2. 概念映像
3. 概念对象

在释义典中，静态概念释义的内容。

根据概念对象来划分，概念标记号的种类
- 永不可知概念对象的概念标记号：“恒”
- 通用概念标记号 = 客观概念标记号 + 人造概念标记号
- 客观概念标记号：客观概念对象的概念标记号，悠忽效果容易被普通人类所识别。
- 人造实物概念标记号：人造实物概念对象的概念标记号，与客观概念对象的概念标记号的悠忽度类似。
- 人造抽象概念对象的概念标记号：际悠忽度最大。
- 指针概念标记号：其字面悠忽度，取决与其指向的目标思想。

1. 原子概念标记号：自由词
2. 分子概念标记号：合成词
3. 宏概念标记号：词组、短语
4. 思想表达
  1. 句子：思想表达的最小单位 
  2. 片段：段落
  3. 选节：小节
  4. 模块：章
  5. 单一或局部思想系统：某个领域的科学理论，主义
  6. 系列思想系统：非单一思想系统。物理理论系统，由物理理论的各个分支组成
  
字面概念释义悠忽度的大小排列顺序
- 自由词 > 合成词 > 词组、短语 > **句子** > 片段 > 选节 > 模块 > 单一或局部思想系统 > 系列思想系统
- 句子是思想表达的最小单位；通用概念标记号只是思想表达的原材料，不构成思想表达。
 
思想表达的最小单位是“句子”。除了“恒”属于难以辩驳的绝对根基参考表达以外，凡是采用概念标记号来表示思想的方法，均属于忽悠表达方式。  
但忽悠度有所不同：
- 指针概念标记号：其字面悠忽度，取决与其指向的目标思想。
- “恒”属于难以辩驳的绝对根基参考表达。
- 人造抽象概念对象的概念标记号悠忽度最大；客观概念对象的概念标记号悠忽度相对要低。实际悠忽度，需要根据实际的语言应用场景来具体问题具体分析。
- Colating 科学极限语言（科极语）又称为”悠忽语“的根本原因，是因为 Colating是一种模糊语言，”模糊“，与之举俱来的副产品就有”忽悠“。由于忽悠既有主观性，也有客观性，所以又称之为“悠忽”。这是为了提醒使用 Colating语言的人们，需要时刻关注由于模糊而导致的“忽悠”现象，以及需要人们自行去解决由于这种“忽悠”现象而造成的不良后果。
