# Colango 概念理论

1. 概念标记号
2. 概念映像
3. 概念对象

在释义典中，静态概念释义的内容。

根据概念对象来划分，概念标记号的种类
- 永不可知概念对象的概念标记号：“恒”
- 通用概念标记号 = 非人造概念标记号 + 人造概念标记号
- 非人造概念标记号：非人造概念对象的概念标记号，悠忽效果容易被普通人类所识别。
- 人造实物概念标记号：人造实物概念对象的概念标记号，与客观概念对象的概念标记号的悠忽度类似。
- 人造抽象概念对象的概念标记号：理论悠忽度最大。
- 指针概念标记号：其概念释义字面悠忽度，取决与其指向的目标思想。

---
1. 原子概念标记号：自由词
2. 分子概念标记号：合成词
3. 宏概念标记号：词组、短语
4. 思想表达
  1. 句子：思想表达的最小单位 
  2. 片段：段落
  3. 选节：小节
  4. 模块：章
  5. 单一或局部思想系统：某个领域的科学理论，主义
  6. 系列思想系统：非单一思想系统。物理理论系统，由物理理论的各个分支组成
  
概念释义字面悠忽度的大小排列顺序
- 自由词 > 合成词 > 词组、短语 > **句子** > 片段 > 选节 > 模块 > 单一或局部思想系统 > 系列思想系统
- 句子是思想表达的最小单位；通用概念标记号只是思想表达的原材料，除了个别概念标记号外，由它们所构成的思想表达，一般悠忽度大；因此，在使用和理解独立应用的通用概念标记号时，需要谨慎。
